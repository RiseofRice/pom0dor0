# pom0dor0

A customizable Pomodoro timer website with session tracking and notes, all stored locally in cookies. The backend is built with **Go**.

## Features

- **Pomodoro Timer**: A fully functional timer for the Pomodoro technique
- **Customizable Settings**: Configure work sessions, break durations, and more
- **Session History**: Track your completed Pomodoro sessions
- **Notes**: Add notes for each session to track what you worked on
- **Cookie Storage**: All data is stored locally in cookies for privacy

## Tech Stack

- **Backend**: Go
